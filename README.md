# BiRAGAS — The Bioinformatics Help Companion

An illustrated, interactive companion to the **BiRAGAS** agentic causal-AI platform:
**51 real bioinformatics questions across 16 themes**, each with a plain-language answer,
its illustration, and the modules that answer it — from a raw FASTQ file to a validated,
FDA-grade causal target.

**Live site:** https://mayasss-gif.github.io/BiRAGAS-Help-Companion/

## What's inside
- Animated hero + capability overview (mirrors [biragas.com](https://biragas.com))
- The companion **film** (captioned / clean toggle) and a downloadable **illustrated brochure (PDF)**
- An interactive gallery of all **51 Q&A**, filterable by theme, by audience, and by live search
- Fully self-contained `index.html` (HTML/CSS/JS) + `assets/` (illustrations, film, brochure)

## Product
The BiRAGAS application lives at **app.biragas.com** — separate from this companion site.
